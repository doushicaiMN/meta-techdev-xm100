const CH *  format,
  ... 
)

Print status message.

Print status message to standard output. The message is printed if level is lower or equal to MSG_LEV2 or verbose is TRUE and level is lower or equal to MSG_DBG; it is discarded otherwise.

Messages assigned a message level MSG_ERR or MSG_WARN are prefixed with "ERROR:" or "WARNING:", respectively. Messages of type MSG_DBG are prefixed with a hyphen. See details for MSG_LEVEL for a definition of valid message levels.

Parameters
levelIN: message level
formatIN: format string
...IN: optional arguments
