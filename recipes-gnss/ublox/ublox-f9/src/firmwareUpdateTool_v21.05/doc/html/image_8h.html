Open file and buffer it in RAM.

Opens a file with platform-independent method (file streams) and copies the content into a dynamically allocated buffer. The pointer to the beginning of the buffer is returned in pFileContent. The caller is responsible of freeing the memory at *ppFileContent if the return value is TRUE.

Parameters
BinaryFileNameFilename of file to open
ppFileContentpointer to buffer to receive file content
pFileSizepointer to file size
Returns
TRUE if the file could be buffered, FALSE else
